# GreenPrompt Dashboard

Two-tab Dash app with company and employee reports.

## Setup
- Create a virtual environment: `python3 -m venv .venv && source .venv/bin/activate`
- Install deps: `pip install -r requirements.txt`

## Run
- Start the server: `python app.py`
- Open the dashboard at `http://127.0.0.1:8050/`

## Contents
- `app.py`: Dash app with company and employee report tabs, charts, and compact KPI tables.
- `requirements.txt`: Python dependencies.
